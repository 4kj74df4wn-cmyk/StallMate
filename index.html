📍 ตลาด
🔒 ความปลอดภัยของข้อมูล
📱 ข้อมูลอยู่ในเครื่องคุณเป็นหลัก
ยอดขาย/เมนู/ตั้งค่าทั้งหมดเก็บในเครื่องก่อนเสมอ ใช้งานได้แม้ไม่มีเน็ต ไม่ต้องพึ่งเซิร์ฟเวอร์กลางเพื่อขายของ
🚫 ไม่เชื่อมบัญชีธนาคาร/บัตร
แอปไม่ขอ ไม่เก็บ และไม่เชื่อมต่อบัญชีธนาคาร บัตรเครดิต หรือรหัสผ่านการเงินใดๆ ทั้งสิ้น
👥 Sync ทีมเห็นเฉพาะรหัสที่ตรงกัน
รหัสเข้าร่วมทีมสุ่มด้วย crypto.getRandomValues (สุ่มปลอดภัยระดับเข้ารหัส เดายากกว่าสุ่มทั่วไปมาก) เครื่องอื่นจะเห็นข้อมูลร้านได้ก็ต่อเมื่อกรอกรหัสตรงกันเท่านั้น
🔑 Master Password กันลบข้อมูลพลาด
ตั้งรหัสผ่านไว้ในหน้าตั้งค่าได้ ระบบจะขอยืนยันทุกครั้งก่อนลบรายการขาย/ลบสินค้าถาวร/ล้างข้อมูล กันพนักงานหรือมือลื่นกดพลาด
🛡️ ฐานข้อมูลกลางมีการป้องกันหลายชั้น
เชื่อมต่อผ่าน HTTPS เข้ารหัสตลอดทาง ตั้ง Security Rules แบบ deny-by-default (ปิดกั้นการอ่าน/เขียนที่ไม่ได้รับอนุญาตทั้งหมด) และจำกัด API Key ให้ใช้ได้เฉพาะจากเว็บแอปนี้เท่านั้น
💾 มีระบบ Backup กันข้อมูลหาย
กด Backup ในหน้าตั้งค่าเพื่อดาวน์โหลดไฟล์สำรองข้อมูลทั้งหมดเก็บไว้เองได้ตลอดเวลา แนะนำทำเป็นประจำโดยเฉพาะก่อนเปลี่ยนเครื่อง

🍎 iPhone / iPad (Safari)
1. เปิดเว็บนี้ด้วย Safari (เบราว์เซอร์อื่นทำไม่ได้)
2. แตะปุ่มแชร์ ⎋ ด้านล่างจอ
3. เลื่อนหาเมนู "Add to Home Screen" (เพิ่มไปยังหน้าจอโฮม)
4. แตะ "เพิ่ม" มุมขวาบน — จะมีไอคอนแอปโผล่ที่หน้าจอโฮมทันที
🤖 Android (Chrome)
1. เปิดเว็บนี้ด้วย Chrome
2. แตะเมนูจุดสามจุด ⋮ มุมขวาบน
3. เลือก "Add to Home screen" หรือ "ติดตั้งแอป" (บางเครื่องมีป๊อปอัพเสนอให้ติดตั้งอัตโนมัติ)
4. ยืนยัน "เพิ่ม"/"ติดตั้ง" — จะมีไอคอนแอปโผล่ที่หน้าจอโฮมทันที
ติดตั้งแล้วเปิดจากไอคอนหน้าจอโฮมได้เหมือนแอปทั่วไป ไม่ต้องเปิดผ่านเบราว์เซอร์อีก
